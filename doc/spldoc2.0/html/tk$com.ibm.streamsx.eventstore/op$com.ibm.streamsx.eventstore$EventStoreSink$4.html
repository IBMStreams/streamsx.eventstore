Connection information

IBMStreams streamsx.eventstore Toolkit > com.ibm.streamsx.eventstore 2.4.1 > com.ibm.streamsx.eventstore > EventStoreSink > Connection information

The operator requires two connection information, one for JDBC and one for the SCALA connection. The following diagram demonstrates how to build the value for the connectionString parameter from the details page of your Event Store instance:



Format for the connectionString parameter: <IP:PORT from JDBC Connection URL>;<SCALA Connection URL> For example: 172.16.213.218:32350;172.16.214.136:30854,172.16.214.49:30854,172.16.214.91:30854
